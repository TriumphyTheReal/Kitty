#
Welcome to KITTY!
## What is KITTY?
KITTY, is a VANILLA CC: Tweaked script that lets your browse websites with a monitor! It does, however, require HTTPS to be Enabled!
# Setup
## Enviroment Setup
Here is how to setup KITTY!
You first need ADVANCED Monitors, not NORMAL monitors. You need them to be in a landscape rectangle, Not a square, ABOVE 3x2.
3x3 will not work.
3x2 will not work.
4x3 WILL work!
You must connect a ADVANCED Computer to the monitor.

## Computer Setup
Now that you have that done. You can copy and run this into your computer:
  `w`
and the rest will be done.

# How to use
When you open your computer, enter in `kitty` and the rest will be on the monitor!
